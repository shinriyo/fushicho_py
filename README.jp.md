FushichoPy
====

Overview

## Description

React code template generator.

## 必要環境

Python 2.7.x
Django 1.6.x

Mac OS X/macOS
Linux
Windows

## 使用方法

```
fushicho.py
```

## Install

```
pip install fushicho
```

## 今後の開発

環境

```bash
virtualenv-3.4 env
```

```bash
source env/bin/activate
```


# ソースパッケージ作成

```bash
python setup.py sdist
```

# パッケージインストール

```bash
python setup.py install
```
